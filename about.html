Allen Miller About

I'm a Performance Test Engineer. Basically, I test our website to see if they can handle a large number of users without issues. If there are issues, it is my responsibiltiy to follow up with respective teams to resolve them. We use Visual Studio Enterprise to create our test scripts and to execute them. Load testing is completely automated and it is non-functional testing.

At my previous job, I worked as a front end developer using Angular. We worked on a call center application for United Airlines and worked on implementing new functionality. I also created various internal applications, including a typeahead search bar, which basically is a search bar that gives suggestions to the user as they type.

Between my two professional jobs, I have recieved about four months total of training, about two months for each job. For SallieMae, we were training in test automation with selenium in both C# and Java, and we also learned some C# backend programming in .NET and Microsoft SQL Server. At Infosys, I was training in the full JavaScript stack, including HTML, CSS, JavaScript itself, Angular, React, NodeJS for a backend, and MySQL. I also learned some Java and Python in college.
